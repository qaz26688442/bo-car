# 松坂搬家 一頁式介紹網站 — 設計文件

- 日期：2026-07-02
- 專案：松坂搬運搬家 ＆ 廢棄物清運 官方一頁式網站
- 狀態：設計已確認，待寫實作計畫

## 1. 目標與定位

為「松坂搬家」建立一個**一頁式（single-page）介紹網站**，讓潛在客人（多為手機使用者）能在一個畫面內快速了解服務、看到實績照片，並用最少步驟直接**撥電話**或**加 LINE** 聯絡詢價。

成功標準：
- 手機開啟載入快、資訊清楚、聯絡按鈕隨手可點。
- 四項服務、實績照片、聯絡方式一頁到底呈現。
- 純靜態檔案，可免費部署、易維護。

## 2. 技術方案

- **純靜態**：單一 `index.html` + 一支 `styles.css` + 一支 `main.js`（僅需少量互動：手機選單、照片燈箱、FAQ 展開、平滑捲動）。
- 無框架、無建置流程、無後端。
- 可部署到 Netlify / Cloudflare Pages / GitHub Pages（免費）。
- RWD 響應式，行動優先（mobile-first）。

理由：一頁式介紹站用框架屬過度設計；純靜態載入最快、最好維護。

## 4. 頁面結構（由上到下）

1. **導覽列 (Header)**：左側公司名「松坂搬家」文字標誌，右側「撥電話」「加 LINE」按鈕；手機版精簡。
2. **Hero 首屏**：橫式實拍大圖為背景（加深色遮罩確保文字可讀），主標「松坂搬運搬家 ＆ 廢棄物清運」、slogan、兩顆主要 CTA（撥電話 / 加 LINE）。
3. **服務項目**：四張卡片，每張一張照片 + 標題 + 簡短說明。
   - 住家搬家、公司/辦公室搬遷、家具/重物搬運、廢棄物清運。
4. **選擇我們的理由**：4 個特色（透明報價／細心包裝／準時到府／搬運＋清運一條龍），用 icon + 短句。
5. **實績照片牆 (Gallery)**：精選數張照片格狀排列，點擊開燈箱放大。
6. **常見問答 (FAQ)**：3–5 題可展開式（accordion）。
7. **聯絡區 / Footer**：再放一次電話、LINE、服務地區、營業時間、公司名。
8. **手機固定浮動列**：畫面底部常駐「撥電話 / 加 LINE」，捲到哪都能點。
